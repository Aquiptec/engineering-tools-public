# Engineering Tools Public

## Purpose

This public repository contains non-confidential engineering calculators, technical tools, and general design support resources developed for water, wastewater, environmental, clean-gas, and process engineering applications.

## Scope

This repository may include:

- General engineering calculators
- Non-client-specific design checks
- Public technical notes
- Generic process engineering tools
- Simple HTML, JavaScript, Python, or spreadsheet-style calculation logic
- Educational or demonstration examples

## Exclusions

This repository must not include:

- Client-specific project files
- Commercial proposals
- Vendor pricing
- Contracts
- Internal correspondence
- Project-specific P&IDs
- Proprietary process calculations
- Confidential design reports
- Personal, financial, or access-control information

## Technical Areas

- Water and wastewater treatment
- Stormwater treatment
- Industrial trade waste
- Clean-gas systems
- Scrubbers and air pollution control
- Process engineering calculations
- Heat and mass balance tools
- Pump, tank, and pipe sizing support
